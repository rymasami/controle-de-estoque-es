# 🗃️ Sistema de Controle de Estoque

Sistema simples de controle de estoque desenvolvido em Python, 
com funcionalidades de entrada e saída de produtos.

Projeto desenvolvido como Atividade Prática da disciplina de 
**Engenharia de Software** — UNINTER (2026).

## 📋 Funcionalidades

- Cadastrar produtos
- Registrar entrada de produtos no estoque
- Registrar saída de produtos com validação de saldo
- Visualizar estoque e histórico de movimentações

## 🛠️ Tecnologias

- Python 3.x

## ▶️ Como executar

```bash
python estoque.py
```
